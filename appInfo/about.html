Physiology Analysis Pipeline

Current Version: 1.2.0
By Scott Harris
09/09/2023

About

The physiology analysis pipeline is built to organize and analyze data from retinal electrophysiology experiments, including patch-clamp recordings. The system is designed to deal with data recorded using Symphony-DAS V2.X.X.X and Stage V2.X.X.X.

The pipeline performs analyses on data from single cells. Batch analyses are not included, but should be straightforward to perform after having used this system to organize single cell data.

Requirements

The following software must be installed on your device in order to run the pipeline:

  • MATLAB 2020+

Note: Clarinet is no longer required as of V1.2.0

Run the pipeline by adding its directory system to your MATLAB path and typing the command >>physiologyPipeline into the MATLAB command window. The pipeline will run on any operating system.
